A helpful bot for the F.A.T. IRC channel
Built with ruby & isaac
-~-> "irc://irc.freenode.net/#fatlab":irc://irc.freenode.net/#fatlab
-~-> "http://fffff.at":http://fffff.at


h2. Commands

!echo <your text here>
Repeat something. See if bot works.

!meme
Spout an automeme from "http://automeme.net/":http://automeme.net/

!hipster
Spout a hipster-flavored automeme

!kanye
Random Kanye West quote via "http://jamiedubs.com/quotable-kanye":http://jamiedubs.com/quotable-kanye

!taco
Gives you a taco

!swineflu
Print the latest H1N1 stats from "Rhiza Labs FluTracker":http://flutracker.rhizalabs.com/

!twitter <msg>
Post msg to a shared twitter account (requires permission)

!search_twitter <keywords>
Print the latest twitter results for a given search

!topic <msg>
Set the channel topic to msg, no ops required (requires permission)

!links
Prints the 3 last-mentioned URLs (stored in memory)
